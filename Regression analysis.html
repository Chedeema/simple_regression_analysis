2. When and How to Use Regression Models

When to Use Regression:
  • Prediction: Use regression when you want to predict the value of one variable based on the value(s) of other variable(s).
  • Understanding Relationships: Use regression to understand the strength and type (positive or negative) of the relationship between variables.
  • Trend Analysis: Use regression to identify trends over time or across various categories.
How to Use Regression:
  • Simple Linear Regression:

When you have one independent variable and want to predict or understand its effect on the dependent variable.

  • Multiple Linear Regression:

When you have multiple independent variables and want to see how they collectively influence the dependent variable.

# Define the independent (X) and dependent (y) variables
X = data['Budget (USD)']
y = data['Conversions']

# Add a constant to the independent variable (for the intercept)
X = sm.add_constant(X)

# Fit the regression model

# Get the summary of the regression model

# Extract the independent variable (Budget) and dependent variable (Conversions)
X = data['Budget (USD)']
y = data['Conversions']

# Coefficients from the regression model
beta_0 = 231.5751  # Intercept (constant)
beta_1 = 0.0033    # Slope (coefficient for Budget)

# Scatter plot of Budget vs Conversions
sns.scatterplot(x=X, y=y)
plt.title('Budget vs Conversions with Regression Line')

# Generate the trend line using the regression equation: Y = beta_0 + beta_1 * X
y_pred = beta_0 + beta_1 * X

# Plot the trend line
plt.plot(X, y_pred, color='red', label='Trend Line')

# Add a legend
plt.legend()

# Show the plot
plt.xlabel('Budget (USD)')
plt.ylabel('Conversions')
plt.show()
